# { title }

Pasek narzędzi znajduje się po lewej stronie mapy i jest sercem aplikacji, ponieważ zapewnia dostęp do głównych funkcji **gpx.studio**.
Każde narzędzie jest reprezentowane przez ikonę i może być aktywowane przez kliknięcie na nią.

<div class="flex flex-row justify-center text-foreground">
    <div>
        <Toolbar class="border rounded-md shadow-lg" />
    </div>
</div>

As with [edit actions](./menu/edit), most tools can be applied to multiple files at once and to [inner tracks and segments](./gpx).

Kolejne sekcje szczegółowo opisują każde z narzędzi.
