# { title }

**gpx.studio** kullanarak GPX dosyalarınızı gösteren haritalar yapabilir ve bunları web sitenizde kullanabilirsiniz.

Tüm ihtiyacınız olan:

1. A <a href="https://cloud.maptiler.com/auth/widget?next=https://cloud.maptiler.com/maps/" target="_blank">MapTiler key</a> to load the map, and
2. GPX dosyanıza erişebileceğiniz bir sunucu veya Google Drive, veya umumi bir URL.

Daha sonra aşağıdaki yapılandırıcı ile haritanızı özelleştirebilir ve HTML kodu oluşturabilirsiniz.

<DocsNote type="warning">

Sunucunuzda <a href="https://developer.mozilla.org/en-US/docs/Web/HTTP/CORS" target="_blank">Cross-Origin Resource Sharing (CORS)</a> kurulu olmalı ve <b>gpx.studio</b> sunucunuzdan GPX dosyalarını indirmesine izin vermeniz gerekmekterdir.

</DocsNote>
