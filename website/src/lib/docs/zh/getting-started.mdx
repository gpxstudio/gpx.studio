# { title }

欢迎来到 **gpx.studio** 官方指南！
本指南将帮助您快速熟悉界面中的所有组件和工具。

<DocsImage src="getting-started/interface" alt="gpx.studio 界面" />

如上图所示，界面存在四个主要的功能区域， 
让我们快速了解一下它们。

## 菜单栏

[菜单栏](./menu) 位于顶部。
可在此处进行常规操作，例如打开、关闭和导出文件，撤消和重做以及其他调整设置。

## 文件和统计

当前已被打开的文件会出现在界面底部。
单击某个文件标签页，列表下方将显示其统计信息。
[文件和统计](./files-and-stats) 章节将介绍如何选择多个文件，并切换到垂直布局以进行高级文件管理。

## 工具栏

[工具栏](./toolbar) 在左侧。包含所有用来编辑文件的工具。

## 地图控件

[地图控件](./map-controls) 位于界面右侧。
包含导航、缩放以及在不同地图样式之间切换等功能。
