使用滑块调整简化算法的距离阈值，查看将保留的轨迹点数量，并在地图中查看简化后的轨迹。

查阅 <a href="https://en.wikipedia.org/wiki/Ramer%E2%80%93Douglas%E2%80%93Peucker_algorithm" target="_blank"> 简化算法文档 </a> 以了解更多。
