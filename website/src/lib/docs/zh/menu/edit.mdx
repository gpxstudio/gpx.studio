# { title }

与文件操作不同，编辑操作可能会修改当前所选文件的内容。
若启用文件 [树形布局](../files-and-stats)，操作也可作用于相关 [轨迹、路段和兴趣点](../gpx)。
在文档中，所有可通过以上操作修改的元素被称为 _文件项_。
注意，除撤消和重做外，其余操作也可通过右键单击文件项来唤出操作菜单。

### <Undo2 size="16" class="inline-block" style="margin-bottom: 2px" /><Redo2 size="16" class="inline-block" style="margin-bottom: 2px" /> 撤销和重做

撤销或重做最后一次执行的操作。
仅适用于在界面上直接进行的所有操作，不适用于显示选项、应用设置或地图导航。

### <Info size="16" class="inline-block" style="margin-bottom: 2px" /> 信息…

用于查看或编辑所选文件项的名称和说明信息。

### <PaintBucket size="16" class="inline-block" style="margin-bottom: 2px" /> 外观…

用于调整所选文件项在地图上显示的轨迹颜色、不透明度和线宽。

### <EyeOff size="16" class="inline-block" style="margin-bottom: 2px" /> 隐藏/显示

切换所选文件项在地图上的可见性。

### <Plus size="16" class="inline-block" style="margin-bottom: 2px" /> 新建轨迹

在所选文件项中创建一条新轨迹。

<DocsNote>

此操作仅在启用树形布局时可用，
且所选内容必须是单个文件。

</DocsNote>

### <Plus size="16" class="inline-block" style="margin-bottom: 2px" /> 新建路段

在所选轨迹中创建一个新路段。

<DocsNote>

此操作仅在启用树形布局时可用，
且所选内容必须是单条轨迹。

</DocsNote>

### <FileStack size="16" class="inline-block" style="margin-bottom: 2px" /> 全选

选中当前文件层级中所有内容。

### <Maximize size="16" class="inline-block" style="margin-bottom: 2px" /> 居中

根据所选文件项调整地图显示中心。

### <ClipboardCopy size="16" class="inline-block" style="margin-bottom: 2px" /> 复制

复制所选文件项到剪贴板。

<DocsNote>

此操作仅在启用树形布局时可用，

</DocsNote>

### <Scissors size="16" class="inline-block" style="margin-bottom: 2px" /> 剪切

剪切所选文件项到剪贴板。

<DocsNote>

此操作仅在启用树形布局时可用，

</DocsNote>

### <ClipboardPaste size="16" class="inline-block" style="margin-bottom: 2px" /> 粘贴

若剪贴板中的内容与当前文件层级兼容，则进行粘贴操作。

<DocsNote>

此操作仅在启用树形布局时可用，

</DocsNote>

### <Trash2 size="16" class="inline-block" style="margin-bottom: 2px" /> 删除

删除所选文件项。
